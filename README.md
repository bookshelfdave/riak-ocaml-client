riak-ocaml-client
=====

Work in progress

This will be a Riak 1.2 Protobuffs-only client.

###Dependencies

* ocamlfind
* Piqi: http://piqi.org/
* Protobuffs: http://code.google.com/p/protobuf/

* OUnit: http://ounit.forge.ocamlcore.org/
